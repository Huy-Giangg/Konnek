# 📱 Konnek - Real-time Messaging App (Java & Firebase)

**Konnek** là ứng dụng nhắn tin thời gian thực được xây dựng trên nền tảng **Android Native (Java)**, kết hợp cùng hệ sinh thái **Firebase**. Dự án hướng tới mục tiêu tối ưu hóa tốc độ xử lý dữ liệu và cung cấp một môi trường giao tiếp an toàn, bảo mật cho người dùng.

---

## ✨ Tính năng chính

* **Real-time Communication:** Gửi và nhận tin nhắn tức thời thông qua **Firebase Realtime Database** hoặc **Firestore**.
* **User Authentication:** Hệ thống đăng nhập bảo mật qua Email, Google bằng **Firebase Auth**.
* **Cloud Messaging (FCM):** Nhận thông báo đẩy ngay cả khi ứng dụng đã đóng, giúp duy trì kết nối liên tục.
* **Media Storage:** Tải lên và chia sẻ hình ảnh, tài liệu thông qua **Firebase Storage**.
* **User Profile & Search:** Tìm kiếm bạn bè qua email/tên và cập nhật thông tin cá nhân.
* **Contact Sync:** Quản lý danh sách bạn bè và hiển thị trạng thái online/offline.

---

## 🛠 Công nghệ sử dụng (Tech Stack)

Sự kết hợp mạnh mẽ giữa lập trình hướng đối tượng và nền tảng đám mây:

* **Language:** Java (Android SDK)
* **Backend as a Service (BaaS):** Firebase
    * *Authentication*: Quản lý người dùng.
    * *Firestore/Realtime Database*: Lưu trữ dữ liệu tin nhắn.
    * *Cloud Messaging*: Hệ thống thông báo.
* **UI/UX:** XML Layouts, Material Design Components.
* **Image Loading:** Glide hoặc Picasso (tùy thư viện bạn dùng).
* **Architecture:** Android Architecture Components (ViewModel, LiveData).

## ✨ Tính năng nổi bật

* **Real-time Chat:** Gửi và nhận tin nhắn tức thì với Firebase Realtime Database/Firestore.
* **Authentication:** Hệ thống đăng nhập, đăng ký bảo mật qua Firebase Auth.
* **Friend Management:** Tìm kiếm người dùng, gửi yêu cầu kết bạn và quản lý danh sách liên lạc.
* **Push Notifications:** Thông báo tin nhắn mới thông qua Firebase Cloud Messaging (FCM).
* **Media Sharing:** Hỗ trợ gửi hình ảnh và quản lý tệp tin qua Firebase Storage.

---

## 🛠 Công nghệ sử dụng

* **Ngôn ngữ:** Java (Android SDK)
* **Backend:** Firebase (Auth, Firestore, Storage, Cloud Messaging)
* **Kiến trúc:** MVC / Android Architecture Components
* **Thư viện hỗ trợ:** Glide (Xử lý hình ảnh), Retrofit, RecyclerView.
